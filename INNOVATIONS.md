# LungAI-TP ResNet创新点说明

## 创新点概览

| 创新点 | 类型 | 原理 | 效果 |
|--------|------|------|------|
| SE通道注意力 | 简单 | 学习哪些特征通道更重要 | 提升特征表达能力 |
| 空间注意力 | 简单 | 学习关注哪些图像区域 | 聚焦细胞核/腺腔等关键区域 |
| 多尺度融合 | 简单 | 融合不同感受野的特征 | 同时捕捉细胞和组织级特征 |
| 病理感知模块 | 高级 | 结合HE染色先验知识 | 提升病理图像特异性 |

---

## 创新点1: SE通道注意力

### 原理
```
输入特征 (B, C, H, W)
    ↓
全局平均池化 → (B, C, 1, 1)
    ↓
全连接层1: C → C/r (压缩)
    ↓
ReLU激活
    ↓
全连接层2: C/r → C (恢复)
    ↓
Sigmoid → 注意力权重 (B, C)
    ↓
原始特征 × 权重 → 输出
```

### 医学意义
- 学习哪些特征通道对诊断更重要
- 例如：细胞核形状通道 > 背景纹理通道

---

## 创新点2: 空间注意力

### 原理
```
输入特征 (B, C, H, W)
    ↓
通道平均: (B, 1, H, W)
通道最大: (B, 1, H, W)
    ↓
拼接: (B, 2, H, W)
    ↓
7×7卷积 → (B, 1, H, W)
    ↓
Sigmoid → 空间注意力图
    ↓
原始特征 × 注意力图 → 输出
```

### 医学意义
- 学习关注图像的哪些区域
- 自动聚焦于细胞核、腺腔等关键区域

---

## 创新点3: 多尺度特征融合

### 原理
```
输入特征 (B, C, H, W)
    ↓
┌─────────┬─────────┬─────────┬─────────┐
│ 1×1卷积 │ 3×3卷积 │ 5×5卷积 │ MaxPool │
│ (像素级) │ (局部)  │ (区域)  │ +1×1    │
└─────────┴─────────┴─────────┴─────────┘
    ↓
拼接 (B, C, H, W)
    ↓
1×1卷积融合 → 输出
```

### 医学意义
- 同时捕捉细胞级（小卷积核）和组织级（大卷积核）特征
- 融合多尺度信息提升诊断准确性

---

## 创新点4: 病理感知模块（高级）

### 原理
```
输入图像 (B, 3, H, W) RGB
    ↓
┌─────────────────────────────────────┐
│ 颜色分离层                          │
│ 学习HE染色的4种成分：               │
│ - 细胞核（紫色）                    │
│ - 细胞质（粉色）                    │
│ - 基质（浅色）                      │
│ - 空白（白色）                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 核浆比检测层                        │
│ 输出：(B, 1, H, W) 核浆比图         │
│ 高核浆比 → 恶性可能                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 纹理分析层                          │
│ 检测细胞异型性、排列模式            │
└─────────────────────────────────────┘
    ↓
融合所有特征 → 病理感知加权图像
```

### 医学意义
- **颜色分离**：模拟病理科医生的染色分析
- **核浆比**：高核浆比是恶性肿瘤的标志
- **纹理分析**：检测细胞异型性和组织结构

## 使用方法

### 训练模型
```bash
# 使用所有创新点
python train_pathology.py --epochs 20

# 不使用病理感知模块（简化版）
python train_pathology.py --no_pathology_module
```

### 可视化创新点效果
```bash
python visualize_attention.py
```
